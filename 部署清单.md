## 修改文件（6个）

## 依赖
- `openai` npm 包（场景图模块需要；服务器若已有 RW 导览图功能则已安装）
  如未装：`npm install openai`

## 环境变量
- `OPENAI_API_KEY`（场景图模块需要，配在 .env）
  未配置时：服务器正常启动，仅场景图功能返回友好报错（不崩溃）

## 部署步骤
```bash
cd ~/worksheet-server
# git pull（推荐，确保所有文件版本一致，不会漏）
git pull origin main

# 或手动确认文件齐全后再重启：
for f in fix_pw4_template fix_blackboard fix_pw3_template pw3_extract pw3_regen generate_image; do
  [ -f "src/$f.js" ] && echo "✓ $f.js" || echo "✗ 缺失 $f.js"
done
# 全部 ✓ 后：
npm install openai   # 若未装
pm2 restart server
pm2 logs server --lines 20   # 确认无报错
```
